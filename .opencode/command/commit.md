Añade los cambios al staging area:
```bash
git add -A
```

Haz el commit con un mensaje que siga el formato `fase N descripcion`, donde:
-`N`es el número de fase
-`descripcion` es un resumen breve de esta fase

```bash
git commit -m "fase N descripcion"
